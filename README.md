# 作品說明

## 作品名稱：
Ben 個人履歷網站

## 前端使用技術：
- 使用 React-Hook 建構開發
- 使用 create-react-app 專案開發
- 使用 react-router-dom 設置路由切換
- 使用 Bootstar 5 進行開發設計 ( 含 RWD 響應式)
- 使用 scss 客製化部分 Bootstrap 樣式
- 使用 swiper.js 套件設計輪播效果
